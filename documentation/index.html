Node.js and NPM

You can download Node.js from https://nodejs.org. NPM comes bundled with Node.js.




Install Npm Packages

You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.




Development server

Run: npm run serve for a dev server. Navigate tohttp://localhost:8080/. The app will automatically reload if you change any of the source files.




Build

Run: npm run build to build the project. The build artifacts will be stored in the build/ directory.




Generate

Run: npm run generate Build the application (if needed), generate every route as a HTML file and statically export to dist/ directory.
